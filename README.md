!!! Приложение коннектится к серверу, на данный моментв конфиге указаны данные моего локального сервера. !!!

В файле main.py все API функции.
В файле config.py настройки для сервера и аутентификации.
В auth/auth_unit.py функции хэширования пароля, проверка хэша пароля, функция создания JWT и декодирования JWT, а также функция возвращающая содержимое JWT.
В auth/ ожидается папка certs/, в которой будут лежать файлы с приватным и публичным ключами для создания JWT
